📝 How to Add a New Component

  1. Copy the component template: component-template.html
  2. Rename it following the pattern: UI-[NUMBER]-[component-name].html
  3. Save it in the /components/ folder
  4. Upload design mockups to /mockups/UI-[NUMBER]-[component-name].png
  5. Add implementation code files to /code/UI-[NUMBER]-[component-name].[ext]
  6. Update the component details in the HTML file
  7. The component will automatically appear in this index
