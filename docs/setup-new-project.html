ğŸ“œ Framework Scripts & Functions

./framework/generators/create-project.js
Creates new project structure, initializes config, sets up GitHub repo
./framework/generators/create-scenario.js
Generates complete scenario with all domains (business, UX, technical), creates folder structure, index files, navigation
./framework/generators/create-user-story.js
Adds new user story to existing scenario, generates USS steps, UX flows, technical components, tests
./framework/generators/create-steps.js
Adds individual steps to existing user story, creates HTML files, GitHub issues, cross-references
./framework/generators/update-sitemap.js
Regenerates sitemap.html with current structure, updates navigation menus across all pages
./framework/generators/create-links.js
Scans all components and creates cross-reference links, updates "Related Components" sections automatically
./framework/generators/migrate-structure.js
Migrates existing projects to new framework structure, preserves content while reorganizing files
./framework/deploy/validate-structure.js
Checks naming conventions, file completeness, validates HTML, links, component references, reports broken links and missing files
./framework/deploy/github-pages.js
Builds optimized version for GitHub Pages, generates static navigation, compresses assets, commits and pushes to gh-pages branch
./framework/utils/clone-component.js
Duplicates component with new ID, updates references
./framework/utils/merge-scenarios.js
Combines multiple scenarios into new master scenario
./framework/utils/sync-github-issues.js
Creates/updates GitHub issues for all components, syncs component status with GitHub project boards
