üí° What is USE?

USE (Unified Scenario Engineering) is built on the proven foundation of BDD (Behavior Driven Development) but extends it with business-friendly language and GitHub-first collaboration.

üîß Technical Foundation

Uses BDD's "Given-When-Then" format and executable scenarios under the hood

üë• Business-Friendly

Uses clear USE terminology: Scenarios, Actors, and Business/Technical/UX domains

üöÄ GitHub-First

Integrates with GitHub Issues, Actions, and Projects for zero-infrastructure collaboration
